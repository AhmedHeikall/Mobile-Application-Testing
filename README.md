## 📌 Project Overview
This project covers **requirement analysis, test scenarios, test case design, test execution, defect reporting, and traceability** for key modules of the Facebook mobile app:

- Adding Posts  
- Adding Stories  
- Messaging  

The project started with requirement gathering and Software Requirement Specification (SRS) creation, followed by deriving test scenarios, designing detailed test cases, organizing test suites, managing tests in Jira Xray, executing test cases, reporting defects, and creating a Requirements Traceability Matrix (RTM).

The objective was to ensure complete requirement coverage, validate application functionality, identify defects, and demonstrate professional manual testing practices used in real-world software projects.

All activities are executed using **Jira with xRay**, and screenshots are taken to ensure evidence-based reporting.

## 📝 Software Requirement Specification (SRS) Summary

### **Introduction**
Facebook is a global social networking service connecting billions of users. This SRS defines the **functional and non-functional requirements** for the mobile application.

### **Functional Requirements**
**FR1: Adding Posts**
- Text-only posts  
- Single/multiple photos, videos, GIFs  
- Camera photo posts  
- Tagging friends  
- Check-ins & locations  
- Adding feelings/activities  
- Live videos  
- Background colors  
- Share to Instagram  
- Draft saving  
- Privacy settings: Public, Friends, Custom, Only Me  
- Posting to albums  

**FR2: Adding Stories**
- Single/multiple stories (photo/video)  
- Camera or gallery posts  
- Tagging people  
- Save story to device  
- Privacy settings: Public, Friends, Custom, Hide from someone  
- Comments & replies  
- Story interactions: mute, share to Instagram  
- Add stickers, text, music, effects, drawing, animation, boomerang, green screen  

**FR3: Messaging**
- Start conversations (single/multiple friends)  
- View previous conversations & online friends  
- Send voice, image, location, emojis  
- Audio & video calls  
- Mute conversations  
- Profile viewing  
- Set theme, emoji, nickname  
- View media, files, links  
- Search in conversation  
- Enable/disable notifications  
- Share, restrict, block, report contacts  

### **Non-Functional Requirements**
- **Security**: SSL encryption, secure storage of credentials  
- **Reliability**: Stable performance across devices  
- **Maintainability**: Easily maintainable system  
- **Portability**: Runs on multiple platforms (Windows, MacOS, iOS, Android)
